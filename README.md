# Conception et développement de Bootability Checker, un outil d’analyse de la “bootabilité” d’un disque: cas de FreeBSD

# Objectifs:
1. Comprendre la séquence de démarrage d’un PC
2. Manipuler les structures bas niveau d’un disque
3. Lire un périphérique bloc 
4. Interpréter les structures pour décider si un disque est bootable.
5. Concevoir un outil robuste (gestion erreurs, options, aide).

# Fonctionnalités attendues
1. Analyse d’un disque spécifié
2. Détection du type de table de partition (BIOS ou UEFI)
3. Rapport synthétique : 
   - BOOTABLE / NON BOOTABLE
   - Mode : BIOS, UEFI, BIOS+UEFI ou Aucun
